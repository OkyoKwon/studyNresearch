# 사용자 입력 정리

## 제품/기능
- **제품명**: 개인 도메인 지식 스터디 사이트
- **유형**: 정적 위키/노트 사이트 (AI가 콘텐츠 생성)
- **대상 사용자**: 본인 1명 (인증 불필요)

## 목표
- 특정 도메인 지식을 지속적으로 축적하고 체계적으로 학습
- 초기 도메인: PG(Payment Gateway) 서비스, MOR(Merchant of Record) 서비스, 가상자산 규제
- 도메인/서비스 영역을 자유롭게 추가 가능한 확장 구조
- 각 도메인별 대표 제품/서비스 정보 포함

## 기술 제약 조건
- **프레임워크**: 선호 없음 (최적 선택 위임)
- **배포**: GitHub Pages
- **콘텐츠 관리**: 마크다운 파일 기반
- **반응형**: 모바일 기기 대응 필수

## 콘텐츠 형태
- AI가 작성하는 위키/노트 형태
- 도메인별 구조화된 지식 정리
- 대표 제품/서비스 카탈로그 포함

## 팀 정보
- 1인 개발 (사용자 본인)
- AI 어시스턴트 활용
